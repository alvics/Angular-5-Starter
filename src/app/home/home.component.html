Bootstrap 4 - Angular 5 starter template

Use this boilerplate as a way to quickly start any new Angular/Bootsrap project. You get 5 pages with some text and mostly
barebones HTML document. This starter comes loaded with jquery, popper.js, bootstrap.js, bootstrap 4, ngx-bootsrap, bootswatch, font-awesome, service-worker, pwa-tools & routing
